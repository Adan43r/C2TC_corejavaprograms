# C2TC Core Java Programs

This repository contains **Core Java programs** developed as part of the *C2TC (Campus to Corporate)* training module.  
It serves as a learning resource for understanding **Java fundamentals**, **OOP concepts**, and **basic application development**.

---

## 📌 Features
- Well-structured Java programs for various topics.
- Covers **OOP concepts**: classes, objects, inheritance, polymorphism, abstraction, encapsulation.
- Demonstrates **Java basics**: data types, control statements, loops, arrays.
- Includes **exception handling**, **collections framework**, and **file handling**.
- Beginner-friendly code with inline comments.

---

## 🛠 Technologies Used
- **Java SE** (JDK 8+)
- **IDE**: Eclipse 
- Git & GitHub for version control
